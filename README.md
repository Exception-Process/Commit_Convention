# Commit Convention

커밋 타입 작성 시 이모지는 필수이며, 글자와 이모티콘 또는 이모지만 작성하여 표현합니다.
커밋 메시지는 영문 명령 형식으로 작성합니다.

## Good
> `🧩feat :: implement login logic` or `🧩 :: implement login logic`

## Bad
> `feat :: implement login logic` or `🧩feat :: login logic`
